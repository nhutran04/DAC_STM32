- Chủ đề: Thực hành DAC của STM32
- Chức năng: Sử dụng DAC của STM32 để xuất tín hiệu ngõ ra 2V.
- Cách sử dụng:
1. Mở file cấu hình (.ioc) để xem cấu hình STM32, Nối các chân đúng cấu hình.
2. Gắn dây nạp STM32.
3. Mở file project trên STM32CubeIDE để compile và chạy chương trình.
4. Dùng đồng hồ vạn năng đo điện áp tại ngõ ra để xem kết quả.
